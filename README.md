📦 Sistema de Controle de Estoque (Java CLI)
Uma aplicação de console simples e eficiente desenvolvida em Java para gerenciar o inventário de produtos. Ideal para quem está começando a organizar fluxos de entrada e saída de mercadorias.

🚀 Funcionalidades
O sistema opera via terminal e oferece as seguintes operações:

Adicionar Produto: Registra o nome do item, a quantidade disponível e o valor unitário.

Visualizar Estoque: Exibe uma lista detalhada de todos os produtos cadastrados, incluindo o valor total em estoque.

Sair: Encerra a aplicação de forma segura.

🛠️ Tecnologias Utilizadas
Linguagem: Java 17+ (ou superior)

Paradigma: Programação Orientada a Objetos (POO)
